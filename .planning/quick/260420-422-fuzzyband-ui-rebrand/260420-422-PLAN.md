# Quick Task 260420-422: fuzzyband UI Rebrand

## Goal

Redesign the plugin UI with a professional, minimal aesthetic inspired by the Pacific NW: dark forest greens, wet stone grays, lichen accents. Rename from "Metal Accompaniment" to "fuzzyband". Match the clean flat look of professional plugins like NadIR.

| Token | Hex | Usage |
|---|---|---|
| bg | #111a10 | Main background |

| text-secondary | #8aaa80 | Diagnostic readouts, version |
| text-dim | #4a6448 | Placeholder, help text |
